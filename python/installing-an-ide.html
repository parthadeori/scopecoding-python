Installing and IDE

Here, in this section we'll be learning to install an IDE. What is an IDE? IDE stands for Integrated Development Environment. An IDE is a code editor, where we can edit, write or view codes. We can use our simple Notepad to write codes, but an IDE makes our life easier as a developer. There are a lot of code editor or IDE available. One of them is called PyCharm which is developed by JetBrains. It is focused mainly on Python. Another IDE which I prefer and is used by most of the professional developers is VS Code which is developed by Microsoft. You can use any other code editor you like. But I'll be using VS Code and will help you to set up VS Code for Python. In VSCode you can write codes for other programming languages if you wish to learn in future. Below I have added two links from where you can download VSCode and PyCharm. Both are available for Windows, Linux and MacOS.

  • VS Code
  • PyCharm (Download the free Community Edition)

Steps to Install VS Code

  • Click on the VS Code link above to go to the official VS Code Page from where you can download VS Code.
    Or just click here: https://code.visualstudio.com .
  • Click on Download for Windows and let the download finish.
  • Once it is downloaded, tick the I accept the agreement box and click Next.
  • Tick the required boxes as shown below and click Next.
  • Now click Install and let the Installation Complete.
  • You have succesfully installed VS Code. Now tick on Lauch Visual Studio Code and click Finish.
